# fcj-DuongThaiThuan Internship Report

Báo cáo thực tập sử dụng Hugo với theme hugo-theme-learn.

## Cấu trúc thư mục

```
├── content/          # Nội dung báo cáo (viết bằng Markdown)
├── themes/           # Theme Hugo
├── config.toml       # Cấu hình Hugo
└── README.md         # File này
```

## Chạy local

```bash
hugo server
```

Truy cập http://localhost:1313

## Deploy

Website được tự động deploy lên GitHub Pages thông qua GitHub Actions khi push lên nhánh `main`.
