### Week 3 Goals:

- Master Serverless services on AWS.
- Practice data storage and build basic APIs.
- Understand how AWS services coordinate in the SmartMenu architecture.
- Prepare the foundation for system design and development.

### Achievements During Week 3:

- Understood data storage mechanisms on Amazon S3 and Bucket management.
- Successfully built and deployed basic AWS Lambda Functions.
- Created and tested REST APIs using Amazon API Gateway.
- Grasped the role of Amazon CloudFront in distributing static content.
- Understood how Serverless services coordinate with each other in a system.
- Established a foundation to transition to the analysis and architecture design phase of SmartMenu in the following week.
