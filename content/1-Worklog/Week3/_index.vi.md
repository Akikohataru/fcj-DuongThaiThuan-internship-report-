### Mục tiêu Tuần 3:

- Nắm vững các dịch vụ Serverless trên AWS.
- Thực hành lưu trữ dữ liệu và xây dựng API cơ bản.
- Hiểu cách các dịch vụ AWS phối hợp trong kiến trúc SmartMenu.
- Chuẩn bị nền tảng cho việc thiết kế và phát triển hệ thống.

### Kết quả đạt được Tuần 3:

- Hiểu cơ chế lưu trữ dữ liệu trên Amazon S3 và quản lý Bucket.
- Xây dựng và triển khai thành công các AWS Lambda Function cơ bản.
- Tạo và kiểm thử REST API bằng Amazon API Gateway.
- Nắm được vai trò của Amazon CloudFront trong việc phân phối nội dung tĩnh.
- Hiểu cách các dịch vụ Serverless phối hợp với nhau trong một hệ thống.
- Có nền tảng để bước sang giai đoạn phân tích và thiết kế kiến trúc SmartMenu ở tuần tiếp theo.
