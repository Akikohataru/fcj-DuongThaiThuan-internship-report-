### Week 4 Goals:

- Learn about services supporting deployment and infrastructure management on AWS.
- Grasp architectural design principles according to the AWS Well-Architected Framework.
- Practice Infrastructure as Code with AWS CloudFormation.
- Complete AWS foundation studies to prepare for the development phase of the SmartMenu system.

### Achievements During Week 4:

- Practiced deploying AWS resources using AWS CloudFormation.
- Grasp how to design and manage databases using Amazon DynamoDB.
- Understood the 6 pillars of the AWS Well-Architected Framework and how to apply them to system design.
- Understood Serverless architecture on AWS and the coordination between Lambda, API Gateway, DynamoDB, and S3.
- Completed the AWS foundational learning phase, ready to transition to the analysis, design, and development phase of the SmartMenu system.
- Finalized documentation and implementation plans for the following weeks.
